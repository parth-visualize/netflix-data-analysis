## 📊 Netflix Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to understand patterns in movies and TV shows available on the platform. The analysis includes data cleaning, preprocessing, and exploration to uncover insights about Netflix content.

## 📦 Dataset
The dataset contains information about Netflix titles including:
- Title
- Type (Movie / TV Show)
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre (listed_in)
- Description

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy

## 📈 Goals of the Analysis
- The objective of this project is to explore:
- Distribution of Movies vs TV Shows
- Content trends over the years
- Popular genres on Netflix
- Country-wise content distribution
- Rating distribution

## 🚀 How to Run the Project
- Clone the repository
- git clone https://github.com/parth-visualize/Netflix-Data-Analysis.git
- Navigate to the project folder
- cd netflix-eda
- Install required libraries
- pip install pandas numpy jupyter
- Open the notebook
- jupyter notebook
- Then run Netflix EDA.ipynb.
